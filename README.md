# IT Solutions & Career Platform

Ин вебсайти корпоративӣ ва платформаи корёбӣ барои ширкати IT мебошад, ки бо тарҳи муосири торик (Dark Mode) ва рангҳои аксентии кабуд сохта шудааст. Лоиҳа аз 5 саҳифаи асосӣ иборат аст.

## 🚀 Саҳифаҳои Лоиҳа (Pages)

1. **Home (Саҳифаи асосӣ):** - Сарлавҳаи асосӣ (Hero section) бо шиори "Transforming Data into Decisions".
   - Бахши шарикон ва хизматрасониҳои калидӣ.
   - Саволҳои кампурсидашаванда (FAQ).
2. **Services (Хизматрасониҳо):** - Рӯйхати пурраи хизматрасониҳои рақамӣ бо нишонаҳо (icons) ва тавсифи онҳо.
3. **Career (Кариера):** - Бахши ҷойҳои кории холӣ (Open Positions) бо филтрҳо.
4. **Career Details (Тафсилоти кор):** - Тавсифи пурраи вазифа (масалан, Web Designer), талаботҳо ва шакли ҳуҷҷатсупорӣ (Application Form).
5. **Contact (Алоқа):** - Шакли алоқа (Contact Form) ва маълумот барои тамос.

## 🛠️ Технологияҳо (Technologies Used)

- **HTML5** — барои сохтори саҳифаҳо.
- **CSS3 / Tailwind CSS** — барои тарҳрезӣ (Styling) ва эффекти торик (Dark theme).
- **JavaScript** — барои интерактивӣ (аккордеони FAQ ва филтрҳо).
